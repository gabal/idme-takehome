# Take Home Interview Exercise Gabriel Balda

## Getting Started

First, install node modules:

```bash
npm install
# or
yarn
# or
pnpm install
```

## Run the Development Server
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## Run Tests
```bash
npm run test
# or
yarn test
# or
pnpm test
```

## Create a New Component Using Template
```bash
npm run generate
# or
yarn generate
# or
pnpm generate
```
